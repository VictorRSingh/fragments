# fragments

Fragments API for BTI525

# How to use

Run

```
npm start

```

This will allow you to run the server on localhost:8080 and be able to received requests on the API

```

npm run dev

```

This will run the server in dev mode using the --watch flag to allow the server to restart automatically on file saves.

```

npm run debug

```

This will allow you run the server in debug mode. Attach in VSCode to be able to set breakpoints.
